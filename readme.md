# Comandos

[Link do repositório](https://github.com/juliosouzam/super_healthy)

```sh
# para iniciar o repositório git
$ git init

# para configurar o meu nome no sistema usei esse comando
$ git config --global user.name "Júlio César"

# para configurar o meu email no sistema usei esse comando
$ git config --global user.email "julio.souzam@gmail.com"

# para adicionar todos os arquivos na stage area
$ git add .

# para realizar o commit
$ git commit -m "<message>"

# adicionei o remote o repositório do github
$ git remote add origin git@github.com:juliosouzam/super_healthy.git

# e para subir os commits
$ git push origin main
```
